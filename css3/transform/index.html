Transform

변환(Transform)은 요소의 좌표를 조작하여, 위치나 크기, 회전, 각도 등을 변경하는 속성입니다. 이동(translate), 확대(scale), 회전(rotate), 기울기(skew), 행렬변환(matrix)가 있습니다.

Transform : none | martix() | translate() | translateX() | translatyY() | scale() | scaleX() | scaleY() | rotate() | rotateX() | rotateY() | skew() | skewX() | skewY() | martix3d() | translate3d() | translateZ() | scale3d() | scaleZ() | rotate3d() | rotateZ() | perspective()

종류 문법 예시 설명
translate translate() transform: translate(10px, 20px) X좌표, Y좌표 이동을 나태냅니다.
translate3d() transform: translate3d(10px, 20px, 30px) X좌표, Y좌표 Z좌표 이동을 나태냅니다.
translateX() transform: translateX(10px) X좌표 이동을 나태냅니다.
translateY() transform: translateY(10px) Y좌표 이동을 나태냅니다.
translateZ() transform: translateZ(10px) Z좌표 이동을 나태냅니다.
scale scale() transform: scale(2,2) 선택한 요소의 X좌표, Y좌표 확대를 설정합니다.
scale3d() transform: scale3d(2,2,2) 선택한 요소의 X좌표, Y좌표 Z좌표 확대를 설정합니다.
scaleX() transform: scaleX(2) 선택한 요소의 X좌표 확대를 설정합니다.
scaleY() transform: scaleY(2) 선택한 요소의 Y좌표 확대를 설정합니다.
scaleZ() transform: scaleZ(2) 선택한 요소의 Z좌표 확대를 설정합니다.
rotate rotate() transform: rotate(10deg, 20deg) 선택한 요소의 X축, Y축 회전를 설정합니다.
rotate3d() transform: rotate3d(10deg, 20deg, 30deg) 선택한 요소의 X축, Y축 Z축 회전를 설정합니다.
rotateX() transform: rotateX(10deg) 선택한 요소의 X축 회전를 설정합니다.
rotateY() transform: rotateY(10deg) 선택한 요소의 Y축 회전를 설정합니다.
rotateZ() transform: rotateZ(10deg) 선택한 요소의 Z축 회전를 설정합니다.
skew skew() transform: skew(10deg, 20deg) 선택한 요소의 X축, Y축 기울기를 설정합니다.
skewX() transform: skewX(10deg) 선택한 요소의 X축 기울기를 설정합니다.
skewY() transform: skewY(10deg) 선택한 요소의 Y축 기울기를 설정합니다.
etc transform-origin transform-origin: 20% 30% 선택한 요소의 X, Y축 기준점를 설정합니다.
transform-style transform-style: preserve-3d 선택한 요소의 스타일을 3D로 설정합니다.
Perspective Perspective:500px 선택한 요소의 원근점 거리를 설정합니다.
Perspective-origin Perspective-origin: 500px 선택한 요소의 원근점 방향을 설정합니다.
backface-visibility backface-visibility: hidden; 선택한 요소의 3D 뒷면의 배경 유무를 설정합니다.
